Task 4. Build a web application
20 points
Build a web application that helps a student calculate their GPA based on the transcript.
Technical Requirements:
The solution must consist of exactly one HTML file. CSS and JavaScript must be embedded inside this file. External
libraries, frameworks, CDNs and additional files are not allowed.
You are allowed to use the AI for this. See the example of the Vibe Coding in Google Classroom.
Detailed Requirements and Points:

Subtask 1. The application should display the list of the courses
4 points
When the page loads, a table must automatically appear on screen containing 4 columns "course", "credit", "grade" and
"passed".
"course" column should contain all your courses (completed, failed, and currently in progress) listed exactly as they
appear in your transcript (see Argus system). At the bottom of this list, add 5 additional courses from your program (your
choice). The additional courses must not duplicate existing courses.
"credit" column: The number of the credits for the given course.
"grade" column: Your grade for the given course.
"passed" column: Passed courses: "Yes", Failed or currently in progress courses: "No". The 5 added courses at the bottom,
assume grade A (100), credit 6, and consider them as passed.

Subtask 2. Program Name
1 point
Your current academic program name must be clearly displayed on screen.

Subtask 3. Information about GPA
5 points
Use AI to translate the following document in English: https://iliauni.edu.ge/uploads/other/82/82542.pdf
The translated document must appear inside a tooltip that opens when the user hovers over or clicks the information
icon.

Subtask 4. "Calculate" Button
5 points
A button labeled "Calculate" must be presented on screen. When clicked, the GPA should be calculated according to the
courses shown on the screen. The credits and points should match your transcript. GPA must be calculated using the
official Ilia State University GPA formula, according to the document provided above.

Subtask 5. "Calculate with Introduction to AI" Button
5 points
A button labeled "Calculate with Introduction to AI" must be presented on screen. When clicked, the GPA should be
calculated according to the courses shown on the screen.
Assume that the student's final score for "Introduction to AI" equals:
current accumulated points + 30 final exam points
Calculate the GPA using this assumed score.
The clear description should be available on the screen to make users understand the goal of the button.
Correctness, usability, code quality and compliance with the requirements will all be considered during grading.

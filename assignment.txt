Task 4. Build a web application
20 points
Build a web application that helps a student calculate their GPA based on the transcript.
Technical Requirements:
The solution must consist of exactly one HTML file. CSS and JavaScript must be embedded inside this file. External
libraries, frameworks, CDNs and additional files are not allowed.
Detailed Requirements and Points:

Subtask 1. The application should display the list of the courses
4 points
When the page loads, a table must automatically appear on screen containing 4 columns "course", "credit", "grade" and
"passed".
"course" column should contain all your courses (completed, failed, and currently in progress) listed exactly as they
appear in your transcript (see transcript_from_argus.txt). At the bottom of this list, add 5 additional courses from your program (your
choice). The additional courses must not duplicate existing courses.
"credit" column: The number of the credits for the given course.
"grade" column: Your grade for the given course.
"passed" column: Passed courses: "Yes", Failed or currently in progress courses: "No". The 5 added courses at the bottom,
assume grade A (100), credit 6, and consider them as passed.

5 points

5 points
